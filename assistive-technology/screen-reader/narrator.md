# Narrator

Narrator is a screen reader that comes built into Windows computers.

Learn the basic controls for testing web content.

To navigate web content with Narrator:

## Start Narrator

To Start or Stop Narrator, press the **Windows key + Ctrl + Enter**

## adjust settings like the voice, relative volume, and speed

Windows key + Ctrl + N

## Narrator "modifier" key (Narrator key)

Both the **Caps Lock** and **Insert** keys. 

To toggle **Caps Lock functionality** while Narrator is running, press **Caps Lock twice**.

## Recommended combination

Narrator and Edge

## Reading

* Ctrl: Stop reading
* ↑/↓: Previous/Next paragraph or item
* ←/→: Previous/Next character
* Ctrl + ↑/↓: Previous/next line
* Ctrl + ←/→: Previous/next word
* Narrator + Tab: Reread current item
* Narrator + +/-: Increase/decrease speech rate

## Interactive Elements

Tab

## reading lines/paragraphs

Arrow keys

## Scan Mode

Caps Lock + Spacebar for element navigation (headings, links, forms with specific keys like 'H' or 'L')

Caps Lock + Arrow Keys to cycle through views (headings, links, etc.)

shortcuts, e.g.: Caps Lock + T for window title or Caps Lock + ~ to set focus
