# VoiceOver

## VoiceOver commands: 

press **VoiceOver (VO) modifier + a letter**

VoiceOver modifier (VO) could be **CAPS LOCK** or **CONTROL + OPTION** key

## VoiceOverHelp: 

**VO + K**

## Move VoiceOver Cursor: 

**VO + Left, Right, Up, Down Arrow** keys to **move cursor** to next item on the screen

## Adjust the Voice

**VO + Shift + Command + right arrow key**

## Select Controls
e.g. checkboxes, radio buttons, show items in pop-up menu, select item in menu

**VO + Space Bar**

## Type in Text Fields

Move cursor to label to learn more about the corresponding text field, type in text into text field, passwords are not spoken aloud.
